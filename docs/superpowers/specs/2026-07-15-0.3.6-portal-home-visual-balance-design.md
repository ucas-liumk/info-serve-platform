# 0.3.6 门户首页视觉平衡补充设计

## 1. 背景与结论

本设计补充首页右上角工具栏方案，处理工具栏变轻后出现的首页视觉重心偏上、统计区下方留白过多，以及模块卡片与统计卡片视觉重量接近的问题。

用户确认采用“整体下移 + 统计区降层级 + 背景柔化”的轻量统一方案。首页内容、六列模块结构、模块插画、统计指标和业务行为均保持不变。本次不引入 Bento 重排、额外装饰光效或新图片资产。

## 2. 设计目标

- 模块卡片和统计区作为同一个内容组整体下移，减少首屏下半部无效留白。
- 模块卡片保持主内容层级，统计区退为辅助数据带，不与服务入口争夺注意力。
- 统一两类卡片的圆角、边框、表面色和阴影体系。
- 让长城背景到浅色页面底部的过渡更早、更柔和，使统计区自然落入安静背景。
- 保持 375px、768px、1024px 和 1440px 下不横向溢出，并避免在移动端增加无意义的首屏空白。

## 3. 垂直节奏

只调整 ModuleGrid 的上边距。StatsBand 继续位于模块区之后，二者现有 24px 间距不变，因此会作为一个整体同步移动。

- 大于 1460px：模块区上边距从 190px 对齐到 192px，纳入 8px 间距节奏。
- 1181–1460px：从 112px 调整为 152px，1440px 视觉验收时整体下移 40px。
- 1024–1180px：从 96px 调整为 112px，整体下移 16px。
- 1023px 及以下：保持现有 64px、48px、40px 等响应式规则，不增加移动端纵向空白。

不得用 transform: translateY() 制造视觉位移；必须使用正常文档流间距，避免命中区域、滚动位置和视觉位置不一致。

## 4. 卡片层级统一

### 4.1 模块卡片

- 保留六列结构、现有尺寸、插画、标题和描述。
- 保持 --ip-shadow-md 静态层级和 hover 时 --ip-shadow-lg 的可点击反馈。
- 卡片表面透明色改用现有 --ip-* 令牌派生的 color-mix()，不得新增十六进制、rgb() 或 rgba() 硬编码颜色。
- 模块卡片继续是完整 button，箭头仅作为视觉方向提示，不新增重复点击控件。
- 补齐清晰的 :focus-visible 焦点环；焦点反馈不得引发布局位移。

### 4.2 统计区

- 保持 24px 的模块区间距、现有指标、顺序和网格响应式行为。
- 阴影从 --ip-shadow-md 降为 --ip-shadow-sm，建立辅助层级。
- 表面色使用令牌派生的半透明 neutral 表面，与模块卡片同色系但透明度更高。
- 统计图标圆形从 52px 收敛为 48px，图标字号同步小幅收敛；数据值和标签字号不变，避免损害读数效率。
- 分隔线降低视觉对比度，但仍使用现有 neutral 令牌派生，保证浅色背景下可辨识。

## 5. 背景衔接

- 保留长城背景图片、定位方式和整体品牌氛围。
- 仅调整首页现有白色渐变的停靠位置，让浅色过渡略早进入模块与统计区域。
- 不新增大面积遮罩面板，不模糊背景图片，不改变图片资产。
- 渐变修改不得增加设计审计中的硬编码颜色计数。

## 6. 动效与可访问性

- hover 抬升只用于可点击模块卡片；统计区不增加 hover 动效。
- 模块卡片和“更多服务”按钮必须具有可见键盘焦点。
- 在 prefers-reduced-motion: reduce 下关闭非必要位移和过渡。
- 保持现有 DOM 顺序和标题语义，不改变统计数据的屏幕阅读顺序。

## 7. 文件范围

允许修改：

- plus-ui/src/views/portal/home/index.vue
- plus-ui/src/views/portal/home/components/ModuleGrid.vue
- plus-ui/src/views/portal/home/components/StatsBand.vue
- 如设计审计指标下降，允许同步更新 plus-ui/design-audit-baseline.json，但禁止上调任何基线。

不修改后端、接口、数据库、路由、模块注册表、图片资产和 /VERSION。

## 8. 验收标准

- 1440px 下模块与统计区相对当前版本整体下移约 40px，统计区下方留白明显减少。
- 模块区与统计区仍保持 24px 间距，六列卡片等宽且无重排。
- 模块卡片是主层级，统计区阴影和图标重量明显更轻。
- 背景由长城图向浅色底自然过渡，无新增大面板或明显色带断层。
- 375px、768px、1024px、1440px 无横向滚动、遮挡和异常首屏空白。
- 模块卡片键盘焦点清晰；reduced-motion 下不执行非必要位移动效。
- Windows 验收服务中首页接口无代理 500，模块入口和统计数据显示行为不变。
