## 任务 2：拆分笔记、交流和文件信息数据逻辑

**文件：**

- 新增 `plus-ui/src/views/portal/resources/components/useResourcePreviewNotes.ts`
- 新增 `plus-ui/src/views/portal/resources/components/ResourcePreviewNotesPanel.vue`
- 新增 `plus-ui/src/views/portal/resources/components/ResourcePreviewDiscussionPanel.vue`
- 新增 `plus-ui/src/views/portal/resources/components/ResourcePreviewInfoPanel.vue`

**实施：**

1. 抽取笔记与公开交流共享的请求、刷新、编辑、删除和可见性切换逻辑。

## 任务 3：实现桌面功能轨与抽屉

**文件：**

- 新增 `plus-ui/src/views/portal/resources/components/ResourcePreviewWorkbench.vue`
- 删除或改造 `plus-ui/src/views/portal/resources/components/ResourcePreviewContextPanel.vue`

**实施：**

1. 实现 56px 功能轨，仅显示我的笔记、交流互动、文件信息。
2. 使用 Iconify 线性图标和 neutral-600 / primary 选中态。
3. 实现默认 392px、动态上限的可调宽抽屉；桌面空间不足时自动覆盖。
4. 实现同项二次点击、切换、关闭、Esc、双击复位与宽度持久化。
5. 保持 `quoteSelection` 暴露契约，引用时打开笔记抽屉并填入内容。
6. 添加 `prefers-reduced-motion` 降级和完整 aria 状态。
